Portfolio van Anthony Vágó

Ik heb mijn portfolio opgedeeld in categorieën waarin ik mijn nieuwe verworven vaardigheden en kennis heb verwerkt.


U kunt ervan uitgaan dat de onderwerpen die hier niet worden genoemd, reeds over voldoende kennis beschikten.

Eén voorbeeld hiervan is het uploaden van sensordata naar de cloud via een API.

Tijdens mijn meewerkstage heb ik waardevolle kennis opgedaan in het gebruik van API's om data naar de cloud te verzenden.

Het verkrijgen van data van sensoren vormt een essentieel onderdeel van mijn specialisatie, Embedded Software Development, gedurende mijn opleiding HBO-ICT.
